# Frontend Agent Instructions

In addition to the `base_agent.md` instructions, frontend agents must:

- **Google Agent Development Kit**: Utilize the Google Agent Development Kit for building agentic features.
- **State Management**: Implement clear state management for the frontend application.
- **Frontend/Backend Separation**: Maintain a strict separation between the frontend logic and backend services.
- **Responsiveness**: Ensure the frontend remains responsive even during long-running agentic tasks.
- **Visual Excellence**: Follow modern web design principles to create a premium user experience (as described in the developer guidelines).
